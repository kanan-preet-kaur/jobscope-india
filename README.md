# JobScope India

An interactive data analytics dashboard for exploring India's job market trends, salaries, skills, and hiring insights using Python and Streamlit.

## Dataset

Indian Job Market Dataset (2025–2026)

## Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Project Status
